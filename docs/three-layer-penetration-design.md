# 三层渗透：知识潜移默化设计规范

## 核心理念

知识不是"看"的，是"用"的。用户不需要主动点击知识卡片——心理学概念通过三层渗透自然融入体验。

## 三层架构

### L1 表面层 — 微文案渗透（始终可见）

创意文本风格嵌入UI元素，1句话级别：

**情绪色盘**：
- 中心默认：「你的大脑正在定位...」
- 选完后：「前额叶已收到信号 ✓」

**气泡标签旁顺口溜**：
- 焦虑 → 「杏仁核在拉警报，心跳加速手心汗」
- 平静 → 「迷走神经慢呼吸，腹侧vagus护平安」
- 愤怒 → 「六秒暂停是良药，冲动决定多后悔」
- 快乐 → 「多巴胺回路亮起来，品味此刻莫等待」
- 悲伤 → 「五羟色胺在低谷，允许悲伤是修复」
- 压力 → 「皮质醇短促帮聚焦，长留身体伤不了」

**进度条文案**：
- Step 1：「定位 · 唤醒你的情绪坐标」
- Step 2：「命名 · 叫出它的名字就驯服了一半」
- Step 3：「溯源 · 找到情绪的导火索」
- Step 4：「书写 · 前额叶最爱的表达方式」

**日记 placeholder**：
- 「说出来，杏仁核就"矮"了」
- 「写下来，前额叶就上线了」

**洞察页标题谐音梗**：
- 趋势图 → 「心情'趋'势」
- 散点图 → 「情'散'意不散」
- 标签分布 → 「标签'签'万语」
- 因子分析 → 「因子'因'你而异」

### L2 交互层 — 操作即学习（记录流程中）

**情绪色盘 → 脑区映射**：
选完色盘后，色盘渐变为简化大脑俯视图，对应脑区高亮：
- 积极高唤醒 → 前额叶+伏隔核（奖励回路）
- 消极高唤醒 → 杏仁核+前扣带回（威胁检测）
- 积极低唤醒 → 腹侧迷走神经（安全连接）
- 消极低唤醒 → 背侧迷走神经（冻结退缩）

**因子卡片翻转**：
拖拽因子到时间线时，因子卡片短暂翻转显示心理学机制：
- 工作压力 → 「HPA轴激活 → 皮质醇↑」
- 人际关系 → 「镜像神经元 → 情绪传染」
- 睡眠不足 → 「前额叶功能↓ → 情绪调节力↓」

**保存完成 → 神经递质动画**：
庆祝粒子改为神经递质样式：
- 积极情绪 → 多巴胺分子（圆+尾）
- 消极情绪 → 皮质醇分子（三角+环）
- 中性情绪 → 血清素分子（波浪线）

### L3 深度层 — 渐进揭示（记录N次解锁）

**解锁规则**：
- 记录3次 → 首页浮动「情绪解剖学」分镜（打油诗版）
- 记录7次 → 记录完成页出现「今日脑科学」漫画条
- 记录14次 → 洞察页解锁「神经科学视角」叠加层

**情绪解剖学分镜内容**（打油诗+科学）：
1. 「杏仁核：情绪的火警铃」
   打油诗：杏仁核小脾气爆，零点零一二秒就报到；前额叶慢六秒到，先停后想不乱跑
   科学：杏仁核是大脑的威胁检测器，能在12毫秒内触发战斗/逃跑反应，而前额叶皮层需要约6秒才能介入调节

2. 「迷走神经：安全的开关」
   打油诗：迷走神经分两段，腹侧安全背侧寒；长呼慢吸腹侧亮，安全感从呼吸来
   科学：腹侧迷走神经激活时产生安全感和社交连接，背侧迷走神经激活时产生冻结和解离

3. 「前额叶：理性的指挥官」
   打油诗：前额叶是总指挥，命名情绪就到位；写写说说它就醒，冲动魔鬼变天使
   科学：前额叶皮层负责执行功能和情绪调节，语言化情绪可以激活前额叶、抑制杏仁核

**今日脑科学漫画条**（记录7次后）：
记录完成后，在引导卡片下方显示一条横向漫画，用4格分镜讲述与当前情绪相关的脑科学知识。

**神经科学视角叠加层**（记录14次后）：
洞察页图表上叠加神经科学解读，如趋势图上标注"这段低谷可能是HPA轴持续激活"。

## 浮动贴纸位置优化

**问题**：当前浮动知识便签可能遮挡核心内容，位置随机导致重叠。

**优化方案**：
1. 首页贴纸只在页面下半部分（最近心情区域下方）出现
2. 贴纸位置基于网格系统，避免重叠
3. 贴纸不遮挡 hero 区域和功能卡片
4. 移动端贴纸缩小并固定在底部区域
5. 贴纸出现延迟增加，等用户浏览完核心内容后再浮现
